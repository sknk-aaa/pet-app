プライバシーポリシー

最終更新：2026年1月1日

1. 収集する情報

まいにちペット（以下「本アプリ」）は、以下の情報を収集します。

  • アカウント情報（メールアドレス、Apple ID または Google アカウント）
  • ペットの記録写真・タイトル・メモ（本人が入力したもの）
  • ペットの基本情報（名前・種類・誕生日など）
  • 「今日のペット」への参加を希望した場合の投稿写真
  • プッシュ通知トークン（通知を許可した場合）

2. 情報の利用目的

  • アプリの基本機能（記録の保存・表示）の提供
  • 「今日のペット」機能の運営
  • プッシュ通知の送信
  • サービスの改善・不正利用の防止

収集した情報を第三者に販売することはありません。

3. データの保管

日記写真はお使いの端末内に保存されます。アカウント情報・投稿データは Supabase（米国）のサーバーに暗号化して保存されます。

「今日のペット」に参加した写真は審査後にアプリ内で公開される場合があります。参加は任意であり、いつでも取り下げ可能です。

4. データの共有

  • Apple Sign-In / Google Sign-In を利用する場合、各社のプライバシーポリシーが適用されます
  • 法令に基づく開示要請があった場合を除き、第三者への提供は行いません

5. データの削除

アプリの設定画面からいつでもアカウントを削除できます。削除すると、すべての投稿データ・アカウント情報が完全に削除されます。端末内の写真は削除されません。

6. お子様のプライバシー

本アプリは13歳未満の方を対象としていません。13歳未満の方の個人情報と認識した場合は速やかに削除します。

7. ポリシーの変更

本ポリシーを変更する場合は、アプリ内またはこのページにてお知らせします。

8. お問い合わせ

プライバシーに関するご質問はアプリ内のサポートよりご連絡ください。
