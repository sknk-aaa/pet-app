よくある質問

  • 写真が保存されない：設定 → 写真と保存 → カメラロールにも保存 をオンにしてください。
  • ログインできない：インターネット接続を確認してから再度お試しください。
  • 通知が届かない：設定 → 通知 → まいにちペット から通知を許可してください。
  • データを引き継ぎたい：ログインすることでデータがバックアップされ、機種変更後も引き継げます。
  • アカウントを削除したい：アプリ内 設定 → アカウント → アカウントを削除 から削除できます。
